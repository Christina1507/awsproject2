# Portfolio Website Hosted on AWS EC2

## Description
This project demonstrates launching and configuring an Amazon EC2 instance
to host a personal portfolio website.

## Technologies Used
- HTML, CSS, JavaScript
- AWS EC2
- Amazon Linux
- Apache Web Server
- SSH

## EC2 Configuration Steps
1. Launched an EC2 instance (t2.micro)
2. Configured security group (SSH, HTTP)
3. Connected using SSH
4. Installed Apache web server
5. Deployed portfolio files to /var/www/html
6. Verified access via public IP

## Outcome
Successfully hosted a portfolio website on a public EC2 instance.
